# Tax-Calculator
 Tax Calculator is a website for the purpose of tax calculation based on a users input. Considering various aspects such as age group and deductions , it calculates the final income after taxes.This is a simple tax calculation website built using HTML, CSS, and JavaScript.
It takes up values by the user has to fill up all the fields with numeric values. If the user input in invalid or negative then it returns 0.
